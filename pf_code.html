Programming

In spare time I write libraries for a medium sized online gaming community mod. The API scripting language is Pawn (previously "Small-C") which is a typeless C-like scripting interface. Aside from this I'm also moderately experienced in C# and JavaScript in which I mostly write utility programs. Below is a list of some of the frameworks and libraries I've written in Pawn followed by some small side projects in node.js.

SIF

(GitHub)
A framework for creating interactive gameplay mechanics for building puzzles or roleplay features.

modio

(GitHub)
A binary file IO library designed specifically for modular designed scripts.

Camera Tool

(GitHub)
A tool for creating in-game keyframed camera sequences.

Texture Tool

(GitHub)
A tool for browsing material IDs and applying textures to objects.

Scavenge and Survive gamemode

(GitHub)
My open source survival, PvP and building gamemode.

Object Loader

(GitHub)
Simple object loading script for parsing ASCII 'CreateObject' lines in files.

SimpleINI

(GitHub)
A basic cache based INI file reader/writer.

Line Segment Generator

(GitHub)
A library for creating line segments with in-game objects.

TextureMap Object Placement (Not a library yet)

(Gist)
Uses a bitmap image to distribute objects across an area on the map.

Markdown to BBCode (node.js, Incomplete)

(Gist)
Converts markdown to SA:MP forum compatible BBCode (used for my topics).

Heatmap (node.js, Incomplete)

(Gist)
Generates a heatmap for the San Andreas map from an ASCII list of XY coordinates.
